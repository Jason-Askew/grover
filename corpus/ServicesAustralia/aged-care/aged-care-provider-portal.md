# Aged Care Provider Portal

The Aged Care Provider Portal (ACPP) is a secure place to access online services. Complete the following steps to log in or register for access.

Aged care providers can use the ACPP to make claims for aged care subsidies and supplements for:

-   home care
-   residential care
-   transition care
-   short-term restorative care.

The portal offers a user-friendly navigation menu and self-service features that allow you to:

-   manage which users can claim on your behalf
-   filter details in claim and event screens
-   securely search for care recipients and events.

Log in to Provider Digital Access (PRODA) to access the portal.

[Log in to PRODA](https://www2.medicareaustralia.gov.au:5447/sso/sps/oidc/rp/HPOSWEB/kickoff/proda1)

You must have cookies and JavaScript enabled in your browser for the portal to work.

For help using the ACPP visit the [Health Professionals Education Resources](https://hpe.servicesaustralia.gov.au/aged-care-provider-portal.html) website.

Register for an individual PRODA account

You need a [PRODA account](/provider-digital-access-proda) to access the Aged Care Provider Portal.

If you don’t have one, [register for an individual PRODA account](/how-to-register-for-individual-proda-account).

Request an A user number

You’ll need an individual A user number. You’ll use this to link your PRODA account to the portal.

If you already have an A user number, you'll need to request portal access.

If you don’t have an A user number, you’ll need to request access. You can either:

-   ask your organisation administrator to submit a request online
-   complete the [Register, amend or remove users for Aged Care Provider Portal (AC004)](/ac004?context=20) form.

Request or renew your Aged Care Provider Portal access

Once you’ve set up your PRODA account, your organisation administrator can give you immediate access to the portal.

Your access to the portal will expire every 12 months. You’ll need to renew your access before it expires. Your service’s organisation administrator can do this for you.

Follow these steps to renew A user access to the portal:

1.  [Log in](https://www2.medicareaustralia.gov.au:5447/sso/sps/oidc/rp/HPOSWEB/kickoff/proda1) to PRODA to access the Aged Care Provider Portal.
2.  Select **Service user payment summary**.
3.  Select the **User ID**.
4.  Go to the **Service access** tab and select **Edit access**.
5.  Select the **Calendar icon** in the end date tab to extend your access.
6.  Select the date for 12 months from the date of request and select **Submit**.

Link to the portal

To link to the portal:

1.  [Log in](https://www2.medicareaustralia.gov.au:5447/sso/sps/oidc/rp/HPOSWEB/kickoff/proda1) to your **Individual PRODA account**.
2.  Select **Aged Care Provider Portal** under **Available services**.
3.  Enter your **Aged care user ID** and press **Submit**.
4.  Accept the **Terms and conditions**.

You’ll see the Aged Care Provider Portal under **My linked services** on the PRODA home page.

The first and last name on your current A user number must match your PRODA account. If they don’t match, you won’t be able to access the portal.

You can use the [Register, amend or remove users for Aged Care Provider Portal](/ac004?context=20) form to update your details.

You can read about [using the ACPP](/using-aged-care-provider-portal?context=20).
