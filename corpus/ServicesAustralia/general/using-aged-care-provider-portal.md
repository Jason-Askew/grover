# Using the ACPP

Using the Aged Care Provider Portal (ACPP).

Aged care providers can use the ACPP to make claims for aged care subsidies and supplements:

-   Support at Home
-   Residential care
-   Transition care
-   Home Care
-   Short-term restorative care.

You can access the ACPP with an individual or organisation Provider Digital Access (PRODA) account. If you donâ€™t have an account, [register for PRODA online](/how-to-register-for-individual-proda-account).

[Log in to PRODA](https://www2.medicareaustralia.gov.au:5447/sso/sps/oidc/rp/HPOSWEB/kickoff/proda1)

For help using the ACPP visit the [Health Professionals Education Resources](https://hpe.servicesaustralia.gov.au/aged-care-provider-portal.html) website.
