# Residential events

Register residential care recipient events and respite care recipient events using the Aged Care Provider Portal (ACPP).

You need to tell us about any changes of circumstances for the people in your care by registering an event. We’ll use this information to decide what payments we make to you on behalf of the care recipient.

You’ll need an individual Provider Digital Access (PRODA) account to access the [Aged Care Provider Portal (ACPP)](/aged-care-provider-portal?context=20). If you don’t have one, you can [register for an individual PRODA account](/how-to-register-for-individual-proda-account).

Register a residential care permanent entry event

To register a permanent entry event, you’ll need to be on a **care recipient profile**.

1.  Search for the care recipient using their **care recipient details**.
2.  Select **Register entry**.
3.  Select **Permanent residential care** from the drop-down field.
4.  As you complete the fields in each step, select **Next**.
5.  When you’ve finished working through each step, select **Submit** to complete the process.
6.  If you’ve successfully filled in all the relevant information, you’ll see a **Successful receipt** screen.
7.  Go to the **Entry summary** to confirm it is now accepted.

Register a residential care extra service entry event

To register an extra service entry event, you’ll need to be on a **care recipient profile**.

1.  Select **Extra service** under the **Circumstances** tab to open the **Extra service summary** screen.
2.  Select **Register extra service**.
3.  Once on the **Register extra service** screen:
    -   Select the extra service start date.
    -   Select the extra service end date (if known).
    -   Select the extra service type.
4.  Check the details you have entered and select **Submit**.
5.  If you’ve successfully filled in all the relevant information, you’ll see a **Successful receipt** screen.
6.  Go to the **Extra service summary** to confirm it is now accepted.

Register a residential care leave event

To register a leave event, you’ll need to be on a **care recipient profile**.

1.  In the **Circumstances** tab select **Leave** to open the **Leave summary** screen.
2.  Select **Register leave**.
3.  Once on the **Register leave** screen:
    -   Select the leave type from the drop-down list.
    -   Select the leave start date.
    -   Select the leave end date (if known).
    -   Check the details you have entered and select **Submit**.
4.  If you’ve successfully filled in all the relevant information, you’ll see a **Successful receipt** screen.

Register a residential care departure event

To register a departure event, you’ll need to be on a **care recipient profile**.

Once you are on the profile:

1.  In the **Circumstances** tab select **Departure** to open the **Departure summary** screen.
2.  Select **Register departure**.
3.  Once on the **Register departure** screen:
    -   Select the departure start date.
    -   Select the departure reason from the drop-down list.
    -   Check the details you have entered and select **Submit**.
4.  If you’ve successfully filled in all the relevant information, you’ll see a **Successful receipt** screen.

Register a respite care entry event

To register an entry event, you’ll need to be on a **care recipient profile**.

1.  Search for the care recipient using their **care recipient details**.
2.  Select **Register entry**.
3.  Select **Respite residential care** from the drop-down field.
4.  Once on the **Register entry** screen:
    -   Select the relevant care type.
    -   Select the entry date.
    -   Select a **Yes** or **No** answer to whether the care recipient has received an award or settlement.
    -   Select the relevant extra service if applicable.
5.  Check the details you’ve entered and select **Submit**.
6.  If you’ve successfully filled in all the relevant information, you’ll see a **Successful receipt** screen.
7.  Go to the **Event summary** to confirm it is now accepted.
