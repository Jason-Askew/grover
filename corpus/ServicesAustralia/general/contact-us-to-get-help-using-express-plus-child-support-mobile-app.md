# Contact us

If you need more help read our online guides or call us.

You can:

-   read our [Express Plus Child Support mobile app](/express-plus-child-support-mobile-app-help-set-up-app) help - Set up app online guide
-   call the [Child Support Enquiry Line](#toolbar-phone).

We’ll ask you some questions. You’ll need to know:

-   your Child Support Reference Number
-   if you’ve used the app in the past
-   if you’re using 3G, 4G or a wireless network (Wi-Fi)
-   if you’re using the latest version of the app
-   the type of device and operating system you're using
-   if you’ve set your phone to a foreign language
-   if you can sign in to your Child Support online account - if not, the date you last signed in.

Contact numbers available on this page.

### Child Support enquiry line

Use this line if you have a question about child support or need to report a change in your circumstances. Let us know if you need an interpreter and we’ll arrange one for free.

[131 272](tel:131272)

Monday to Friday 8:30 am to 4:45 pm

Alternative number

[1800 241 272](tel:1800241272)

There are other ways you may want to [contact us](/contact-us).
