# Software for Medicare Online, ECLIPSE and the AIR

Find developers that provide software for Medicare Online, Electronic Claim Lodgement and Information Processing Service Environment (ECLIPSE) and the Australian Immunisation Register (AIR).

We’ve made changes to our digital health channels. You’ll need web services software to access our systems.

## Software developers who provide web services ready software

These software developers have web services ready software for the functions listed. It doesn’t include all certified software developers. We’ve only listed the ones that have agreed to be published. We can’t guarantee the accuracy or completeness of the information.

We do not recommend or endorse any particular software developer and we recommend you undertake your own assessment to determine which software developer’s product will best suit your needs.
