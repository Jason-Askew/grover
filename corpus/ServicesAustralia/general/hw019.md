# Application for a Medicare provider number and/or prescriber number for a medical practitioner form (HW019)

Use this form to apply for an initial or additional Medicare provider number or a prescriber number, or both.

How to use our forms

You must [download our forms and fill them in](/how-to-use-our-forms) so they are processed quickly. You can also electronically sign some of our forms. We have help available if you use assistive technology or need an interpreter.

Download and complete the [Application for a Medicare provider number and/or prescriber number for a medical practitioner form](/sites/default/files/2025-12/hw019-2512en-f.pdf "PDF document").

You can also apply for Medicare Provider Numbers using Health Professional Online Services (HPOS).

When applying online, you’ll need a [PRODA](/provider-digital-access-proda) account to use [HPOS](/hpos).

Learn about [Applying for additional Medicare Provider Numbers](/apply-for-additional-provider-numbers) and [How to apply for a provider number if you’re a medical intern](/medical-intern-application-for-provider-or-prescriber-number).

Make sure you submit your application and supporting documentation to us before your proposed location start date.
