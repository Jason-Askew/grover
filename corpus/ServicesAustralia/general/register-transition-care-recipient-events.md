# Transition care events

You can register transition care events using the Aged Care Provider Portal (ACPP).

You’ll need an individual Provider Digital Access (PRODA) account to [access the Aged Care Provider Portal (ACPP)](/aged-care-provider-portal?context=20). If you don’t have one, you can [register for an individual PRODA account](/how-to-register-for-individual-proda-account).

[Log in to PRODA](https://www2.medicareaustralia.gov.au:5447/sso/sps/oidc/rp/HPOSWEB/kickoff/proda1)

You need to tell us about any changes of circumstances for the people in your care by registering an event. We’ll use this information to decide what payments we make to you on behalf of the care recipient.

Register a transition care event

To register a transition care entry, you’ll need to be on a **care recipient profile**.

1.  Search for the care recipient using their care recipient details.
2.  Once the search result has loaded with the care recipient profile, select the **Register entry** button.
3.  Once the **Register entry** screen has loaded, you will need to:
    -   Select the care recipient’s hospital admission date.
    -   Select their entry date into care.
    -   Enter their functional capacity score on entry.
    -   Check the details you have entered and press **Submit**.
4.  If you’ve successfully filled in all the relevant information, you’ll see a **Successful receipt** screen.

Register a departure event

To register a departure event, you’ll need to be on a **care recipient profile**.

1.  Under the **Circumstances** tab select **Departure** to open the **Departure summary** screen.
2.  Select the **Register departure** button.
3.  Once the **Register departure** screen has loaded:
    -   Select the departure start date.
    -   Select the departure reason from the drop-down list.
    -   Enter the functional capacity score on departure.
    -   Check the details you have entered and select **Submit**.
4.  If you've successfully filled in all the relevant information, you will see a **Successful receipt** screen.

Register short-term restorative care leave events

To register a short-term restorative care leave event you will need to be on a **care recipient profile**.

1.  Under the **Circumstances** tab select **Leave** to open the **Leave summary** screen.
2.  Select the **Register leave** button
3.  Once the **Register leave** screen has loaded:
    -   Select the leave type from the drop-down list.
    -   Select the leave start date.
    -   Select the leave end date.
    -   Check the details you have entered and select **Submit**.
4.  If you've successfully filled in all the relevant information, you will see a **Successful receipt** screen.
